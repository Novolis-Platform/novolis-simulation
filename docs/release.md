# Release

Packages publish from [novolis-simulation](https://github.com/Novolis-Platform/novolis-simulation) via GitHub Packages (`2026.1.*`).

## Versioning

Follow the shared Novolis platform release train. Breaking API changes require a coordinated bump across consuming repos.

## Documentation

Packable libraries enforce strict XML API documentation (`CS1591`) and ship `README.md` on NuGet.
